# Car Management API

## Install dependency

```bash
# NPM user
npm install
```

## config env variable

dont forget to config env

## run sequelize

```
sequelize db:create (if the database does not exist)
sequelize db:migrate
sequelize db:seed:all
```

## run server

```
npm start
```
